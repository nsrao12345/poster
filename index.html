🔧 Firebase Setup Required — One-time only
1. Go to console.firebase.google.com → Create project → Enable Realtime Database + Authentication (Email/Password)
2. In Authentication → Add users for each committee member. For admin: add a user, then in Realtime Database create path /admins/<uid>: true
3. Paste your Firebase config below and click Save. The banner disappears forever.

🔑 Firebase Config (paste from Firebase Console → Project Settings → Your Apps)

Config is saved in your browser only. Never share this page's localStorage with others. For production, host on Firebase Hosting where the config is already embedded.
